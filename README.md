
# Alpeaks - Know your Alps !

## 
Alpeaks is a mountain peaks quizz game. Current features:
- List a predefined chains
- Custom zone
- Built-in peaks list from [OSM](https://www.openstreetmap.org/) API (Alps & Vosges)
- Local high scores
